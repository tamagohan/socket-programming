# Network programming with Rust

## How to use

* start TCP Server

  ```shell
  cargo run tcp server <address>:<port>
  # example
  cargo run tcp server 127.0.0.1:33333
  ```

* start TCP Client

  ```shell
  cargo run tcp client <address>:<port>
  # example
  cargo run tcp client 127.0.0.1:33333
  ```
